> Swagger UI with ExpressJS

## Tools

- Install `pnpm`

  ```bash
  npm install -g pnpm
  ```

## Setup

- Clone the repository

  ```bash
  git clone https://github.com/hardingadonis/express-swagger.git
  ```

- Install dependencies

  ```bash
  pnpm i
  ```

- Start the server

  ```bash
  pnpm start
  ```

- Open the browser and navigate to `http://localhost:3000/swagger`
